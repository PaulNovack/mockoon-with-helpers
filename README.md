### mockoon with helpers
Install dependancies

<code>npm install --save-dev @mockoon/cli handlebars handlebars-helpers</code>

### Helpful commands

#### Start

<code>
sudo ./startMockoon.sh
</code>

Kill Server

<code>
sudo kill -9 $(lsof -ti :3001)
</code>`
